# Installation

## Requirements

- Python >= 3.10

## Basic

```bash
pip install omnipath-metabo
```

## With web service

```bash
pip install "omnipath-metabo[server]"
```

## Install from source

```bash
pip install git+https://github.com/saezlab/omnipath-metabo.git
```

Or for development:

```bash
git clone https://github.com/saezlab/omnipath-metabo.git
cd omnipath-metabo
pip install -e ".[dev,tests]"
```
